These below steps will get you SPI_UART project running on your local PC.
1.Download Mini58 v3.00.003 BSP/Sample code from web address https://github.com/OpenNuvoton/Mini58BSP 
2.Put SPI_UART file on \Mini58BSP-master\SampleCode\StdDriver directory
3.Enter \Mini58BSP-master\Mini58BSP-master\SampleCode\StdDriver\SPI_UART\KEIL directory and open mini58_SpiToUart.uvproj to run SPI_UART project.
There is binary code in SPI_UART\bin folder. User can load Mini58_SpiToUart.bin to Mini58 via tool directly.
